# AutoReconLite

**AutoReconLite** is an advanced Bash-based reconnaissance and vulnerability scanning framework built for penetration testing practice and automation.

It automates common pentesting reconnaissance tasks, providing structured reports, modular design, and an easy-to-use command-line interface.

---

## Features

- TCP SYN scanning
- Service detection
- OS fingerprinting
- Vulnerability scanning with Nmap NSE scripts
- Automated report generation
- Multiple scan modes: `quick` or `full`
- Modular Bash scripting with argument parsing
- Color-coded output for terminal readability
- Logs all scans for auditing
